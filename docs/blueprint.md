# **App Name**: Heidy Vega Music

## Core Features:

- Artist Bio: Display artist bio information
- Music Player: Stream music via Spotify player
- Photo Gallery: Showcase photo gallery of artist
- Concert Dates: Provide upcoming concert dates or subscription signup
- Social Media Links: Display links to artist's social media accounts
- Contact Form: Enable a contact form for inquiries

## Style Guidelines:

- Background color: Off-white (#F9F9F9) for a clean and modern feel.
- Text color: Almost black (#1A1A1A) for high readability.
- Accent color: Dark terracotta (#A94D4D) for buttons, links, and highlights to convey passion and elegance.
- Headline font: 'Great Vibes' script font for the artist's name/logo to add personality.
- Body font: 'Montserrat' sans-serif font for a modern and readable style.
- Use circular social media icons that subtly zoom or change color on hover in the accent color, positioned as a fixed widget on the bottom right.
- Single-page scroll layout with a sticky header for easy navigation.
